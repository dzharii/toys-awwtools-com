# MeetTime

MeetTime is a shareable meeting timeline that shows one UTC meeting time in two participant time zones, with an animated visual countdown toward the meeting point.

## Highlights

- Edit the meeting title, duration, participant names, time zones, and meeting time.
- Update the meeting from UTC or either participant's local time.
- Switch between light, neutral, dark, and automatic themes.
- Share the full meeting configuration through the page URL.
