cwget: wget package manager for C

Download single-file C libraries with ready-to-run commands, install scripts, and minimal sample programs. No gatekeeping. No lock-ins. .nojekyll. No central registry. Just pure anarchy.
Configure your setup once, then fetch libraries with a single (and gloriously long) command line.
Even packages outside the cwget catalog can be installed directly via wget or curl.
Total freedom. Zero restrictions. Full anarchy, by design (LICENSE).
