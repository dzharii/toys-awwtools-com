Content Extraction Demo Page

Page 1 - Native And Semi-Native Forms

This standalone page contains realistic native and semi-native form examples for testing extraction of labels, values, selected options, repeated sections, readonly summaries, disabled controls, and validation-related form content. Each example is visually separated and prefilled with plausible fictional data.
