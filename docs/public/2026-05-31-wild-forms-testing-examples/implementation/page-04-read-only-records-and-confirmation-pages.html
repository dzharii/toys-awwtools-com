Extraction Test Surface

Page 4 - Read-Only Records And Confirmation Pages

This page contains 22 realistic read-only records and confirmation pages built as a testing surface for a content extraction tool. Each example exercises extraction of identifiers, statuses, repeated record rows, timelines, masked values, price totals, and policy text from static document-like layouts. All data is fictional.
