# Rejected Options

## Full Starter Workflow Configurator

Rejected for v1 because run/test/clean targets and target dispatch increase generated-code complexity and validation burden.

## Cross-Platform Decision Matrix

Rejected for v1 because it can imply portability guarantees that the static generator cannot prove.

## Pure Tutorial Or Code-Lens Tool

Rejected for v1 because it risks becoming educational content instead of a copy-first utility.

## Preset Gallery

Deferred because it is useful only after one executable-focused template is correct and fixture-tested.

## Windows-First Starter

Deferred because platform-specific polish should not replace clear platform assumptions in v1.
