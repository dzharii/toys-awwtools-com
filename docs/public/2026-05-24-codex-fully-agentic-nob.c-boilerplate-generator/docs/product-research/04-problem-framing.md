# Problem Framing

The product is a static, browser-only `nob.c` starter generator for C developers adopting `nob.h`. It reduces the blank-page problem by turning common project-shape decisions into a live, editable build-script preview.

It serves first-time adopters through safe defaults and short inline explanations, while serving power users through dense, explicit controls that map directly to generated C code.

The product does not replace `nob.h`, hide the build model, discover dependencies, manage packages, compile code, inspect local repositories, or become a general project IDE.

The first milestone should produce one high-quality artifact: a ready-to-edit `nob.c` for small executable-oriented C projects. The product is judged by whether the generated file is accurate, readable, transparent, deterministic, and safe to copy into a repository.
