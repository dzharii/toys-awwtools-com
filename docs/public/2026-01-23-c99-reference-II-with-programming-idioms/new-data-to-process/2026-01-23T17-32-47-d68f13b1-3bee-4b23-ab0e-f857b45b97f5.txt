Split a space-separated string
URL: [https://programming-idioms.org/idiom/49/split-a-space-separated-string](https://programming-idioms.org/idiom/49/split-a-space-separated-string)
Description: Split a mutable string into space-separated tokens using the standard C tokenizer.

```c
#include <string.h>
#include <stddef.h>

size_t split_spaces(char* s, char** tokens, size_t max_tokens) {
    size_t count = 0;
    char* token = strtok(s, " ");

    while (token && count < max_tokens) {
        tokens[count++] = token;
        token = strtok(NULL, " ");
    }
    return count;
}
```

Join a list of strings
URL: [https://programming-idioms.org/idiom/53/join-a-list-of-strings](https://programming-idioms.org/idiom/53/join-a-list-of-strings)
Description: Concatenate an array of strings into a single buffer using a fixed separator.

```c
#include <string.h>
#include <stddef.h>

void join_strings(
    const char** parts,
    size_t count,
    const char* separator,
    char* output,
    size_t output_size
) {
    output[0] = '\0';

    for (size_t i = 0; i < count; ++i) {
        if (i > 0) {
            strncat(output, separator, output_size - strlen(output) - 1);
        }
        strncat(output, parts[i], output_size - strlen(output) - 1);
    }
}
```

Extract file content to a string
URL: [https://programming-idioms.org/idiom/58/extract-file-content-to-a-string](https://programming-idioms.org/idiom/58/extract-file-content-to-a-string)
Description: Read the entire contents of a file into a dynamically allocated string using standard C I/O.

```c
#include <stdio.h>
#include <stdlib.h>

char* read_file_to_string(const char* path) {
    FILE* file = fopen(path, "rb");
    if (!file) {
        return NULL;
    }

    fseek(file, 0, SEEK_END);
    long size = ftell(file);
    rewind(file);

    char* buffer = (char*)malloc((size_t)size + 1);
    if (!buffer) {
        fclose(file);
        return NULL;
    }

    fread(buffer, 1, (size_t)size, file);
    buffer[size] = '\0';

    fclose(file);
    return buffer;
}
```

Binomial coefficient "n choose k"
URL: [https://programming-idioms.org/idiom/67/binomial-coefficient-n-choose-k](https://programming-idioms.org/idiom/67/binomial-coefficient-n-choose-k)
Description: Compute the binomial coefficient using an iterative integer method.

```c
unsigned long long binomial(unsigned int n, unsigned int k) {
    if (k > n) {
        return 0;
    }
    if (k > n - k) {
        k = n - k;
    }

    unsigned long long result = 1;
    for (unsigned int i = 1; i <= k; ++i) {
        result = result * (n - k + i) / i;
    }
    return result;
}
```

Compute GCD
URL: [https://programming-idioms.org/idiom/74/compute-gcd](https://programming-idioms.org/idiom/74/compute-gcd)
Description: Compute the greatest common divisor of two non-negative integers using the Euclidean algorithm.

```c
unsigned int gcd(unsigned int a, unsigned int b) {
    while (b != 0) {
        unsigned int r = a % b;
        a = b;
        b = r;
    }
    return a;
}
```

Compute LCM
URL: [https://programming-idioms.org/idiom/75/compute-lcm](https://programming-idioms.org/idiom/75/compute-lcm)
Description: Compute the least common multiple of two non-negative integers using GCD.

```c
unsigned int lcm(unsigned int a, unsigned int b) {
    if (a == 0 || b == 0) {
        return 0;
    }
    return (a / gcd(a, b)) * b;
}
```
