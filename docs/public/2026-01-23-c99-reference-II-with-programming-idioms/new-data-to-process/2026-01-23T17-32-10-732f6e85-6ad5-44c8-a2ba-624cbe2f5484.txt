Quine program
URL: [https://programming-idioms.org/idiom/182/quine-program](https://programming-idioms.org/idiom/182/quine-program)
Description: Output the program source code itself without reading from external files.

```c
#include <stdio.h>

int main(void) {
    const char* s =
        "#include <stdio.h>\n\n"
        "int main(void) {\n"
        "    const char* s = \"%s\";\n"
        "    printf(s, s);\n"
        "    return 0;\n"
        "}\n";
    printf(s, s);
    return 0;
}
```

Check if any value in a list is larger than a limit
URL: [https://programming-idioms.org/idiom/191/check-if-any-value-in-a-list-is-larger-than-a-limit](https://programming-idioms.org/idiom/191/check-if-any-value-in-a-list-is-larger-than-a-limit)
Description: Scan an array and execute a callback if any element exceeds a given threshold.

```c
#include <stddef.h>
#include <stdbool.h>

bool any_greater_than(const int* array, size_t count, int limit) {
    for (size_t i = 0; i < count; ++i) {
        if (array[i] > limit) {
            return true;
        }
    }
    return false;
}
```

Return fraction and exponent of a real number
URL: [https://programming-idioms.org/idiom/204/return-fraction-and-exponent-of-a-real-number](https://programming-idioms.org/idiom/204/return-fraction-and-exponent-of-a-real-number)
Description: Decompose a floating point value into its normalized fraction and exponent.

```c
#include <math.h>
#include <stdio.h>

void print_fraction_exponent(double value) {
    int exponent;
    double fraction = frexp(value, &exponent);
    printf("%f %d\n", fraction, exponent);
}
```

Formula with arrays
URL: [https://programming-idioms.org/idiom/208/formula-with-arrays](https://programming-idioms.org/idiom/208/formula-with-arrays)
Description: Apply an element-wise mathematical formula to multiple arrays of equal length.

```c
#include <math.h>
#include <stddef.h>

void compute_formula(
    double* a,
    const double* b,
    const double* c,
    const double* d,
    size_t count,
    double e
) {
    for (size_t i = 0; i < count; ++i) {
        a[i] = e * (a[i] + b[i] * c[i] + cos(d[i]));
    }
}
```
