ChatGPT Tool Reference Sheet

1. Web Search Tool (`web`)

Capabilities

  • Retrieve up-to-date internet information
  • Access local, location-specific data
  • Extract content from specific URLs

Use Cases

  • News updates: "Search the web for the latest on AI safety."
  • Local: "Find events in NYC this weekend."
  • URLs: "Open this URL and summarize it."
  • Live data: "Check AAPL stock price."
  • Policies: "Is the remote work policy still active?"

2. Python Tool (`python`)

Capabilities

  • Run calculations, analyze data, visualize
  • Process files: CSV, Excel, JSON
  • Build charts, perform regressions, simulate models

Use Cases

  • Math: derivatives, integrals, statistics
  • Data: filter, group, summarize
  • Plots: bar, line, scatter, histograms
  • Files: transform Excel/CSV/JSON
  • Modeling: regression, t-tests
  • Series: trend, seasonal decomposition
  • ML: basic classifiers, regression
  • Text: frequency, summarization
  • Matrix: inverse, eigenvalues
  • Simulate: Monte Carlo, sampling
  • Finance: IRR, NPV, amortization
  • Geo: calculate distances

3. Image Generation Tool (`image_gen`)

Capabilities

  • Generate images from text prompts
  • Edit uploaded images: objects, backgrounds, styles

Options

  • Prompt: description of scene
  • Size: 1024x1024, 1024x1792, 1792x1024
  • Variations: up to 4 outputs
  • Transparency: for logos/icons
  • Reference image: guide generation or edits

4. Text Editing Tool (`canmore`)

Capabilities

  • Create/edit documents or code persistently
  • Enable structured iteration and collaboration

File Types

  • Document: Markdown, essays, manuals
  • Code: Python, HTML, JS, React, Java

Use Cases

  • Write: articles, reports, guides
  • Develop: code, UIs, utilities
  • Document: APIs, setups, changelogs
  • Revise: edit by pattern, section, or full rewrite

5. Guardian Tool (`guardian_tool`)

Capabilities

  • Deliver U.S. state-specific voter/election info

Use Cases

  • Register: check deadlines, eligibility
  • Vote: early, absentee, mail-in, in-person
  • Polls: locate stations, hours, ID needs
  • Ballot: view candidates, measures
