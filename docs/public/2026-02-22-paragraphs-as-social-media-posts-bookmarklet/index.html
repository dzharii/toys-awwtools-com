Paragraph Social Bookmarklet

Drag the button below to your bookmarks bar. Then open a long-form article and click the bookmark to reframe its paragraphs as card-style posts with reactions and share links.

Drag to Bookmarks

If your bookmarks bar is hidden, enable it in your browser UI, then drag again.

How it works

This page loads a readable JavaScript file that defines bookmarklet_paragraph_social_mode. It then generates a bookmarklet URL using bookmarklet_paragraph_social_mode.toString() and assigns it to the install link.

The bookmarklet source is intentionally readable (not minified). Some browsers enforce bookmarklet size limits, so diagnostics below help debug install failures.
