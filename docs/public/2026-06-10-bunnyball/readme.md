# Bunny Volleyball

A calm backyard scene where three rabbits endlessly pass an animated beach ball while an ambient clock tracks local time and visible watching time.

## Highlights

- Seeded randomized passes with varied arcs, speed, shadows, trails, and contact effects
- Editable rabbit names, pass progress, audio preferences, and watching time saved locally
- Optional procedural Web Audio music and sound effects that start only after user activation
- Responsive desktop, landscape, and portrait layouts with reduced-motion support
- Dependency-free HTML, CSS, JavaScript, inline SVG, and local assets

Open `index.html` directly or serve the folder from any static host, including GitHub Pages.
