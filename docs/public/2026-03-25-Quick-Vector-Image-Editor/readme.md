# Quick Vector Image Editor

Desktop-oriented browser editor for fast screenshot annotation and lightweight image composition.

Highlights:

- SVG is the canonical editable format and includes editor metadata for reopen-and-revise.
- PNG, JPEG, SVG export, plus Copy PNG when the browser supports clipboard image writes.
- IndexedDB autosave with retained snapshots, stale-write prevention, and recovery feedback.
- Verbose in-app activity logging to help debug import, autosave, open, and export failures.

This project is intentionally narrow in scope: text, rectangle, ellipse, line, and embedded raster images for quick practical edits.
