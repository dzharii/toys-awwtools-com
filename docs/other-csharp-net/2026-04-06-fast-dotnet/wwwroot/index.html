Localhost Tool Starter

LocalFastUi

Small ASP.NET Core shell for browser-based utilities on localhost HTTPS. Open the local URL, confirm the API, then replace the starter workspace with your own interface.

This page is structured as starter content, not a finished demo. Keep the shell if it helps, or swap sections out one by one.

Open starter workspace View API reference

Edit these files

  • wwwroot/index.html controls the page structure.
  • wwwroot/styles.css holds the starter UI classes.
  • wwwroot/app.js wires buttons to the local API.

Starter classes in use

  • lf-row groups buttons and compact controls.
  • lf-field pairs a label with an input or textarea.
  • lf-output gives API responses a stable reading surface.

Copyable section pattern

Use this small pattern when you add a new content block and want it to match the page immediately.

<section class="lf-section">
  <div class="lf-section-head">
    <h2>Your section title</h2>
    <p>One sentence explaining what belongs here.</p>
  </div>

  <div class="lf-surface lf-stack">
    <label class="lf-label" for="example-input">Example input</label>
    <input class="lf-input" id="example-input" type="text" />
  </div>
</section>

API reference

Compact route and command reference for the local server.

Method Route Use
GET / Starter UI and live browser-side checks.
GET /api/ping Smallest health-style endpoint. Returns plain text ok.
GET /api/info Returns environment, framework, OS, and process details.
POST /api/do Echo-style sample endpoint for request-body handling.
